How to export your files

CLIO

Clio Tasks (1 or 2 CSV files)

Go to Tasks → click Columns and make sure these are visible: Status, Name, Description, Due Date, Assigned To, Assigned By, Matter, Priority → Export as CSV.

You can export Outstanding and Completed separately or together — the dashboard reads the Status column to tell them apart automatically.

LT

Law Toolbox — Date Range Report (CSV)

Go to Reports → Date Range Report → select date range → Download CSV.

Upload all files at once in any order. Each file is auto-detected by its column structure, and Clio tasks are merged and classified by their Status field (pending / complete).
